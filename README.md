# Banking Management System

A secure and responsive Banking Management System built with **React**, **Vite**, and **React Router**.

## Features
- **Secure Login**: Access the system with admin credentials.
- **Dashboard**: Overview of banking services.
- **Account Details**: View profile, account number, and balance.
- **Transaction History**: Track all recent banking activities.
- **Fund Transfer**: Transfer funds securely to other accounts.
- **Protected Routes**: Ensures that only authenticated users can access sensitive pages.

## Technologies Used
- **Frontend**: React (Hooks, Router)
- **Styling**: Vanilla CSS
- **Build Tool**: Vite

## Getting Started
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
